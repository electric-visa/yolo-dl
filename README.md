# YOLO DL <br> 
## A graphical macOS frontend for yle-dl 

### What is YOLO DL?

YOLO DL is a graphical frontend for **[yle-dl](https://github.com/aajanki/yle-dl)**. <br> 
It is aimed at users without the technical knowledge required to install and run yle-dl from the command line. 

YOLO DL comes bundled with all the dependencies needed to run yle-dl, and it can be installed from a .dmg

The app is written in Swift, utilizing SwiftUI and a bit of AppKit.

YOLO DL is licensed under [GPLv3](https://www.gnu.org/licenses/gpl-3.0.html), honoring the copyleft requirement.

### System Requirements: <br> 
YOLO DL requires macOS 14 Sonoma or newer. <br>
The app might work on older macOS versions, but it has not been tested.
